1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, subcategoria Ensino Médio. O jogo aborda o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando de forma divertida a importância do cuidado com a agricultura e da proteção das plantações.
No jogo, o jogador controla um robô agricultor que deve coletar sementes para ganhar pontos e evitar pragas que prejudicam a produção. O projeto busca mostrar como a tecnologia e a sustentabilidade podem trabalhar juntas no campo.


2 - Objetivo
O objetivo deste projeto é conscientizar os usuários sobre a importância da agricultura sustentável, incentivando o cuidado com a produção agrícola e o combate às pragas de forma educativa e interativa.
Público-alvo
O projeto é destinado a estudantes, professores e comunidade em geral interessados em compreender a importância do equilíbrio entre a produção agropecuária, a tecnologia e a preservação ambiental.
